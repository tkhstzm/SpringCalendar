# カレンダーメモ管理アプリ

## 概要
個人開発で作成した、カレンダーに連動したメモ管理用のWebアプリケーションです。
訓練校で学んだ内容をもとに、Java・Spring Boot・MVCモデルの理解を深めることを目的として制作しました。

## 使用技術
- Java / Spring Boot
- Thymeleaf
- BootStrap
- PostgreSQL
- HTML / CSS

## 主な機能
- カレンダー表示
- 日付ごとのメモ登録・編集・削除
- メモ内容の入力チェック（バリデーション）

## 工夫した点
- MVC構成を意識して実装
- バリデーション処理を実装

## 起動方法
1. PostgreSQL にデータベースを作成
2. application.properties に接続情報を設定
3. Spring Boot アプリケーションを起動
